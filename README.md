# Lamborghini Miura SV 1972 Showcase

A luxurious, modern single-page website showcasing stunning photographs of a 1972 Lamborghini Miura SV in rich brown with gold wheels.

## 📸 Image Setup

Add your 4 high-resolution images to the `public/images/` folder with these exact names:

1. `front-view.jpg` — Front three-quarter view in front of classical building
2. `scissor-doors.jpg` — Side view with scissor doors open
3. `hood-headlight.jpg` — Close-up front hood and headlight
4. `rear-view.jpg` — Rear three-quarter view

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Open browser
# http://localhost:3000
```

## 🎨 Features

- **Hero Section**: Full-screen hero with elegant text overlay
- **Gallery**: Clean 3-4 column grid with smooth hover effects
- **Lightbox Modal**: Beautiful image viewer with smooth transitions
- **Details**: Miura SV specs, engine details, and history
- **Responsive**: Optimized for desktop, tablet, and mobile
- **Dark Theme**: Luxurious dark background with gold accents

## 🛠️ Tech Stack

- **Next.js 14** — Modern React framework
- **Tailwind CSS** — Utility-first CSS
- **TypeScript** — Type-safe development
- **Next.js Image** — Optimized image loading

## 🎯 Design Philosophy

Luxurious minimalist dark theme with gold accents, serif headings (Playfair Display), clean sans body text (Inter). Smooth scrolling, elegant animations, and cinematic feel throughout—designed to feel like a high-end automotive art gallery.
